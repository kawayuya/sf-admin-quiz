## トラブルシューティング

### ビルドエラー

| エラー | 解決方法 |
|--------|---------|
| Pod install failed | `cd ios && rm -rf Pods && pod install && cd ..` |
| Code signing failed | Xcode で Signing Certificate を確認・再生成 |
| Memory issues | メモリ使用量を削減、不要なファイルを削除 |

### App Store 審査での却下

| 却下理由 | 対応方法 |
|---------|---------|
| プライバシーポリシーが不明確 | プライバシーポリシーを詳細に記載 |
| テスト用アカウント情報が不足 | App Review Notes に詳細なテスト手順を記載 |
| クラッシュやバグ | TestFlight でテストして修正 |
| ガイドライン違反 | App Store Review Guidelines を確認 |
