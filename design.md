# SF Admin Quiz - デザイン計画

## アプリ概要

**SF Admin Quiz** はSalesforce認定アドミニストレーター試験の対策用モバイルアプリ。120問の過去問から20問をランダムに出题し、即座にフィードバックを提供します。カテゴリ別の成績分析により、弱点を可視化し、効率的な学習をサポートします。

## 画面一覧

### 1. ホーム画面 (Home)
- アプリロゴ＋タイトル
- 「クイズを始める」ボタン（大きく目立つCTA）
- 過去の成績サマリー（最高スコア、平均スコア）
- カテゴリ別弱点バッジ（前回の弱点カテゴリを表示）

### 2. クイズ画面 (Quiz)
- 進捗バー（問1/20 など）
- 問題文エリア（カード形式）
- カテゴリラベル（例：「ユーザー管理」）
- 選択肢リスト（3〜5択、タップで選択）
- 「回答する」ボタン
- 回答後：正解/不正解のインライン表示＋解説テキスト
- 「次の問題へ」ボタン

### 3. 結果画面 (Result)
- スコア表示（例：「18/20 正解」）
- 円グラフ or スコアリング視覚化
- カテゴリ別正解率テーブル
- 弱点カテゴリのハイライト（赤/オレンジ）
- 「もう一度挑戦」ボタン
- 「ホームへ戻る」ボタン

### 4. 統計画面 (Stats) ※タブ
- 過去の受験履歴リスト（日時・スコア）
- カテゴリ別累計正解率グラフ
- 弱点カテゴリランキング

---

## キーユーザーフロー

### メインフロー（クイズ）
1. ホーム画面 → 「クイズを始める」タップ
2. クイズ画面：問題表示 → 選択肢タップ → 「回答する」タップ
3. 即時フィードバック（正解/不正解 + 解説）
4. 「次の問題へ」タップ → 次問題へ（2〜3を繰り返し）
5. 20問終了 → 結果画面へ自動遷移
6. スコア・弱点確認 → 「もう一度」or「ホームへ」

---

## レイアウト方針

- **1画面1タスク**：問題文と選択肢のみ、余計な情報を排除
- **大きなタップターゲット**：選択肢ボタンは最低48px高さ
- **進捗の可視化**：上部プログレスバーで残り問数を常に表示
- **即時フィードバック**：回答後に色変化（緑/赤）とアイコンで直感的に伝達
- **ボトムタブ**：「クイズ」「統計」の2タブ構成
